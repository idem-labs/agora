# Ágora Web

Dashboard for exploring open data catalog health and quality.

## Features

- Catalog overview: registered catalogs, dataset counts, sync status
- Dataset explorer: quality scores, usage stats, resource health
- Health dashboard: uptime trends, failure rates, alerts
- Search: explore the curated index directly from the browser

## Planned stack

Next.js + Tailwind + shadcn/ui
