# Ágora Platform

Backend service that aggregates open data quality intelligence.

## Features

- **Telemetry ingestion**: receives usage signals from MCP clients (opt-in)
- **Quality scoring**: 5-dimension scoring engine (accessibility, structure, freshness, usability, completeness)
- **Automated pipelines**: catalog crawling, resource health checks, periodic re-scoring
- **REST API**: quality scores, catalog registry, analytics
- **Auth & billing**: freemium tiers (free / pro / enterprise)
